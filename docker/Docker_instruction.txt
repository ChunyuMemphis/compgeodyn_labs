Setting up a Linux development environment using Docker
0. Install Docker and create an account on dockerhub.com
1. Start a docker terminal(#1)
2. Start another docker terminal (#2) and run 'docker login' in #2.
3. Run on #1: docker run -ti -v ${HOME}:/home/${USER} ubuntu:14.04 bash
4. Run on #1: adduser ${USER}
5. Run on #1: adduser ${USER} sudo
6. Run on #1: su ${USER}
7. Install any other packages you need on #1
e.g., sudo apt-get -y install gcc 
      sudo apt-get -y install g++
      sudo apt-get -y install gdb
      sudo apt-get -y install make
      sudo apt-get -y install git
      sudo apt-get -y install libboost-all-dev
      sudo apt-get -y install man
      sudo apt-get -y install emacs
8. In #2, run 'docker ps' and get the container ID.
9. In #2 run: docker commit ${CONTAINER_ID} TAG, where TAG can be something like ${DOCKER_HUB_ID}/dev_env:init
10. In #2, run: docker push ${DOCKER_HUB_ID}/dev_env:init
11. Repeat 7-10 as necessary, adding a proper tag to each new image.
12. Later you can start a "container" by running one of the images (typiclly the latest) you've created and start working as if you were on a linux machine.
